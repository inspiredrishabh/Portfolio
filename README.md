## ğŸ”§ Installation & Setup

### Prerequisites

- Node.js (v16.0.0 or newer)
- npm or yarn

### Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/inspiredrishabh/Portfolio/
   cd portfolio
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   yarn
   ```

3. Start development server:

   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Build for production:
   ```bash
   npm run build
   # or
   yarn build
   ```
